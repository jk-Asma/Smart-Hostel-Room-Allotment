# Smart Hostel Room Allotment System

A web-based application that automates the hostel room allocation process using data analysis and machine learning techniques. The system groups students based on their preferences and characteristics to improve roommate compatibility and reduce manual effort in hostel management.

Project Overview

Traditional hostel room allocation is often performed manually, making the process time-consuming and prone to errors. The Smart Hostel Room Allotment System provides an intelligent solution by collecting student information, analyzing preferences, and automatically assigning rooms using clustering algorithms.

The project aims to make room allocation more efficient, transparent, and fair while minimizing administrative workload.


## Features

-Student information collection through forms

-Automated room allotment process

-Preference-based student grouping

-Admin login and management panel

-Data processing using machine learning techniques

-CSV file upload and management

-Downloadable allotment reports

-Responsive and user-friendly interface

## System Workflow

1. Students submit their information through the google form .


2. Student data is stored and processed.


3. Data is cleaned and prepared for analysis.


4. Machine learning algorithms group students with similar preferences.


5. Rooms are allocated based on generated clusters.


6. The admin reviews and manages the allotment results.


7. Final room allocation reports can be downloaded.
